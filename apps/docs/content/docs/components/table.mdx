## Usage

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableRow,
  TableHead,
  TableCell,
  TableCaption,
} from '@/components/ui/table';
```

```tsx
<Table>
  <TableHeader>
    <TableRow>
      <TableHead>Name</TableHead>
      <TableHead>Status</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Alice</TableCell>
      <TableCell>Active</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

## Examples

### With custom cell content

Pass React elements instead of strings to render anything inside a cell:

```tsx
import Badge from '@/components/ui/badge';

<TableCell>
  <Badge label="Active" variant="default" />
</TableCell>
```

### Fixed column widths

```tsx
<TableHead style={{ flex: 2 }}>Name</TableHead>
<TableHead style={{ flex: 1 }}>Role</TableHead>
```

### Without footer

```tsx
<Table>
  <TableHeader>
    <TableRow>

    ))}
  </TableBody>
</Table>
```

## API Reference

### Table

Wraps everything in a horizontal `ScrollView` so wide tables scroll naturally.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional styles for the inner root view. |

### TableHeader

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional styles. Has a bottom border by default. |

### TableBody

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional styles. |

### TableFooter

Renders with a subtle tinted background and a top border.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional styles. |

### TableRow

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle \| (state) => ViewStyle` | — | Additional styles. Supports pressed state. Has a hairline bottom border. |

### TableHead

Column header cell.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional cell container styles. |
| `textStyle` | `TextStyle` | — | Additional text styles (applied when children is a string/number). |
| `children` | `string \| number \| ReactNode` | — | String/number children are auto-wrapped in a styled `Text`. |

### TableCell

Data cell.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional cell container styles. |
| `textStyle` | `TextStyle` | — | Additional text styles (applied when children is a string/number). |
| `children` | `string \| number \| ReactNode` | — | String/number children are auto-wrapped in a styled `Text`. |

### TableCaption

Centered caption below the table. Maps to a native `Text`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `TextStyle` | — | Additional text styles. |
