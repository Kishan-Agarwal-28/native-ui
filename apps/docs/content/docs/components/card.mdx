## API Reference

### Card

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"default" \| "sm"` | `"default"` | Padding and border-radius size. `"sm"` uses `spacing.md` / `radius.lg`; `"default"` uses `spacing.lg` / `radius.xl`. |
| `style` | `ViewStyle` | — | Additional container styles. |

### CardHeader

A row container that places its children between a `CardTitleGroup` and an optional `CardAction`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional styles. |

### CardTitleGroup

Wraps `CardTitle` and `CardDescription` in a flex column.

### CardTitle

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `TextStyle` | — | Additional text styles. |

### CardDescription

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `TextStyle` | — | Additional text styles. |

### CardAction

Non-shrinkable slot for an action button or icon in the card header.

### CardContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional styles. |

### CardFooter

A right-aligned row for action buttons, separated from the content by a top border.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | Additional styles. |
