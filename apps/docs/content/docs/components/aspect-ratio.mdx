## Usage

```tsx
import AspectRatio from '@/components/ui/aspect-ratio';
```

```tsx
<AspectRatio ratio={16 / 9}>
  {/* your content */}
</AspectRatio>
```

## Examples

### Square crop

```tsx
<AspectRatio ratio={1} style={{ width: 200 }}>
  <Image source={{ uri: '...' }} style={{ flex: 1 }} resizeMode="cover" />
</AspectRatio>
```

```tsx
<AspectRatio ratio={4 / 5} style={{ width: '100%' }}>
  <Image source={{ uri: '...' }} style={{ flex: 1 }} resizeMode="cover" />

<AspectRatio ratio={16 / 9} style={{ width: '100%', borderRadius: 12, overflow: 'hidden' }}>

```

## API Reference

### AspectRatio

This is a direct re-export of `@rn-primitives/aspect-ratio` Root. See the [RN Primitives docs](https://rnprimitives.com/aspect-ratio/#props) for the full API.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `ratio` | `number` | `1` | Width-to-height ratio (e.g. `16/9`, `4/3`, `1`). |
| `style` | `ViewStyle` | — | Additional styles applied to the outer container. |
| `children` | `ReactNode` | — | Content to constrain. |
