```tsx
import Spinner from '@/components/ui/spinner';
```

```tsx
<Spinner />
```

## Examples

### All sizes

```tsx
<Spinner size="sm" />
<Spinner size="default" />
<Spinner size="lg" />
```

### Custom numeric size

```tsx
<Spinner size={64} />
```

### Custom color

```tsx
<Spinner color="#8b5cf6" />
```

### Custom thickness

```tsx
<Spinner size="lg" thickness={4} />
```

### Full-screen loading overlay

```tsx
<View style={StyleSheet.absoluteFillObject, { justifyContent: 'center', alignItems: 'center', backgroundColor: 'rgba(0,0,0,0.3)' }}>
  <Spinner size="lg" color="white" />
</View>
```

### Inside a button (manual loading state)

```tsx
<View style={{ flexDirection: 'row', alignItems: 'center', gap: 8 }}>
  <Spinner size="sm" color="white" />
  <Text style={{ color: 'white' }}>Saving…</Text>
</View>
```

### Centered in a screen

```tsx
<View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
  <Spinner size="lg" accessibilityLabel="Loading your feed" />
</View>
```

## API Reference

### Spinner

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "default" \| "lg" \| number` | `"default"` | Size of the spinner. Named sizes map to theme spacing: `sm` → `spacing.lg` (16px), `default` → `spacing.xl` (20px), `lg` → `spacing.xl * 2` (40px). Pass a number for a custom pixel size. |
| `color` | `string` | `colors.primary` | Color of the spinning arc. Defaults to the theme primary color. |
| `thickness` | `number` | `size * 0.1` (min 2) | Border width of the spinner ring. |
| `accessibilityLabel` | `string` | `"Loading..."` | Screen reader announcement. |
| `style` | `ViewStyle` | — | Additional styles. |

> The spinner has `accessibilityRole="progressbar"` and `accessibilityState={{ busy: true }}` set automatically.
