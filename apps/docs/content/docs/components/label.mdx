## Usage

```tsx
import Label from '@/components/ui/label';
```

```tsx
<Label>Username</Label>
```

## Examples

### Required field

Appends a destructive-colored asterisk and includes "Required field" in the screen reader label.

```tsx
<Label required>Password</Label>
```

### Disabled state

```tsx
<Label disabled>Unavailable field</Label>
```

### Required and disabled

```tsx
<Label required disabled>
  Locked setting
</Label>
```

### With custom styles

```tsx
<Label style={{ fontSize: 16, color: '#8b5cf6' }}>
  Custom Label
</Label>
```

## API Reference

### Label

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `required` | `boolean` | `false` | Appends a `*` in `colors.destructive` and adds "Required field" to the accessibility label. |
| `disabled` | `boolean` | `false` | Reduces opacity to 50% and signals "Disabled" to screen readers. |
| `style` | `TextStyle` | — | Additional text styles. |
| `accessibilityLabel` | `string` | Auto-generated | Override the computed accessibility label. |
| `children` | `string \| ReactNode` | — | Label content. |
