```tsx
import Separator from '@/components/ui/separator';
```

```tsx
<Separator />
```

## Examples

### Horizontal (default)

```tsx
<View>
  <Text>Section A</Text>
  <Separator />
  <Text>Section B</Text>
</View>
```

### With a label

When a `label` is provided, it is centered between two lines and announced to screen readers as a separator landmark.

```tsx
<Separator label="OR" />
```

### Vertical

```tsx
<View style={{ flexDirection: 'row', height: 40, alignItems: 'center' }}>
  <Text>Left</Text>
  <Separator orientation="vertical" />
  <Text>Right</Text>
</View>
```

### Custom thickness and spacing

```tsx
<Separator thickness={2} spacing={24} />
```

### Section divider with label

```tsx
<Separator label="Continue with" spacing={20} />
```

## API Reference

### Separator

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Direction of the separator line. |
| `label` | `string` | — | Text centered between the lines. Only available for horizontal orientation. When omitted the separator is fully decorative and hidden from the accessibility tree. |
| `thickness` | `number` | `1` | Pixel thickness of the line. |
| `spacing` | `number` | `16` | Vertical margin (horizontal) or horizontal margin (vertical) around the separator. Also controls the horizontal padding of the label. |
| `style` | `ViewStyle` | — | Additional container styles. |
