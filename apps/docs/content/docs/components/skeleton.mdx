## Examples

### Text lines

```tsx
<View style={{ gap: 8 }}>
  <Skeleton style={{ height: 16, width: '90%' }} />
  <Skeleton style={{ height: 16, width: '75%' }} />
  <Skeleton style={{ height: 16, width: '85%' }} />
</View>
```

### Card skeleton

```tsx
<View style={{ gap: 12, padding: 16 }}>
  {/* Avatar + title row */}
  <View style={{ flexDirection: 'row', alignItems: 'center', gap: 12 }}>
    <Skeleton style={{ width: 48, height: 48, borderRadius: 24 }} />
    <View style={{ flex: 1, gap: 8 }}>
      <Skeleton style={{ height: 14, width: '60%' }} />
      <Skeleton style={{ height: 12, width: '40%' }} />
    </View>
  </View>
  {/* Content lines */}
  <Skeleton style={{ height: 14, width: '100%' }} />
  <Skeleton style={{ height: 14, width: '80%' }} />
  {/* Action button */}
  <Skeleton style={{ height: 40, width: '100%', borderRadius: 12 }} />
</View>
```

### Avatar skeleton

```tsx
<Skeleton style={{ width: 56, height: 56, borderRadius: 28 }} />
```

### Conditional rendering

```tsx
{isLoading ? (
  <View style={{ gap: 8 }}>
    <Skeleton style={{ height: 20, width: '70%' }} />
    <Skeleton style={{ height: 16, width: '50%' }} />
  </View>
) : (
  <View>
    <Text style={{ fontSize: 18, fontWeight: '700' }}>{data.title}</Text>
    <Text>{data.subtitle}</Text>
  </View>
)}
```

## API Reference

### Skeleton

Accepts all standard `ViewProps`. Size and shape are fully controlled via `style`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `style` | `ViewStyle` | — | **Set `width` and `height` here.** Also accepts `borderRadius` for circular or pill shapes. |

> The component automatically adjusts its background color for light (`#d4d0d054`) and dark (`#201f1f82`) themes. The pulse animation runs infinitely with a 1000ms ease-in-out cycle.
